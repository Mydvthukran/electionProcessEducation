# electionProcessEducation (Election Guide Assistant)

A smart civic education assistant that helps users understand elections with clear, step-by-step guidance.

## Features
- Topic detection + structured answers
- Optional Google Gemini integration
- Offline/rule-based fallback
- Accessible, mobile-first UI

## Tech stack
- React + Vite
- CSS (responsive + accessibility)
- Optional Google Generative AI (Gemini)

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
